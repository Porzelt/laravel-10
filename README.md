# Laravel 10 — Projeto de Estudo e Referência

Repositório de estudo e experimentação com Laravel 10, utilizado como base 
de referência durante a migração de sistemas legados PHP para arquitetura Laravel 
moderna na PMESP.

## Stack
- PHP 8.x / Laravel 10
- MySQL / MariaDB
- Docker + Docker Compose (ambiente local via WSL)
- APIs RESTful

## Objetivo
Consolidar padrões modernos de desenvolvimento: OOP, MVC, SOLID, 
Eloquent ORM, migrations, seeders e boas práticas de Clean Code 
aplicadas em sistemas reais de alta disponibilidade.
